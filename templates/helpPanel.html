Usage

  • Log in by clicking the login button at the top right
  • Enter your username and password and click log in
  • Now on the calendar, coloured dots indicate if a request exists on this day and its status
    • Green: request confirmed by all users
    • Orange: Request is still pending for one or more users
    • Red: One or more users have denied this request
  • Clicking on a day will take you to the day information page
  • If a request exists for this day, you can see the information, and edit if it is your request
  • From this page you can change your status (confirm/deny) for someone else's request
  • If no request exists on the day, you can create one by clicking the "Make Request" button and entering a description
  • Most of this info is available if not logged in, but no settings can be changed
  • There is a user page, to change your password, set or change password, and edit notification setting (if email is set)
